Privacy Policy

Effective date: April 7, 2026  ·  Last updated: April 7, 2026

The short version: Neutralize does not collect, store, or sell any personal data. The only external network request made is sending your highlighted text to the Anthropic API to perform analysis.

What Neutralize does

Neutralize is a browser extension that analyzes selected text on webpages for rhetorical techniques (such as emotional appeals, loaded language, and structural devices) and provides a neutral rewrite. Analysis is powered by the Anthropic Claude API.

Data we collect

We collect no personal data. Neutralize does not:

  • Collect your name, email, or any account information
  • Track your browsing history or the websites you visit
  • Store any text you highlight on webpages
  • Use cookies or any form of persistent tracking
  • Send analytics or usage data to any server

Your API key

Neutralize requires you to provide your own Anthropic API key. This key is:

  • Stored locally on your device only, using your browser's built-in storage.local API
  • Never transmitted to any server other than Anthropic's own API endpoint (api.anthropic.com)
  • Never shared with the extension developer or any third party

Text sent to Anthropic

When you highlight text and trigger an analysis, the following information is sent directly from your browser to the Anthropic API:

  • The highlighted text you selected
  • The title and domain of the page you are on (e.g., "nytimes.com")

This data is sent directly from your browser to Anthropic — it does not pass through any servers controlled by the extension developer. Anthropic's use of this data is governed by their Privacy Policy and Usage Policy.

Third-party services

Neutralize uses one external service:

  • Anthropic API — to perform rhetorical analysis. See Anthropic's Privacy Policy.

No other third-party services, analytics platforms, or advertising networks are used.

Permissions

Neutralize requests the following browser permissions:

  • storage — to save your API key locally on your device
  • scripting / content scripts — to detect text selections on webpages
  • tabs — to read the title of the current page for context in the analysis
  • sidePanel (Chrome only) — to display the results panel alongside the page

Children's privacy

Neutralize does not knowingly collect any information from children under 13. No personal information is collected from any user, regardless of age.

Changes to this policy

If this privacy policy changes in a material way, the effective date above will be updated. Because Neutralize collects no personal data, most changes will relate to new external services or permission additions — both of which would be clearly described here.

Contact
